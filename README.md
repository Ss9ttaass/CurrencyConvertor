Данный сервис выполняет функцию конвертера валют на основании данных курсов валют Национального Банка Республики Беларусь. 
В основе сервиса лежит библиотека FastAPI. Для хранения данных используется SQLite с использованием ORM SQLAlchemy.
К особенностям данного решения можно отнести следующее:
- Автоматическая идемпотентная подгрузка курсов валют НБ РБ при запуске решения один раз в 09:00 утра;
- Присутсвует возможность подгружать актуальные курсы валют в любое время;
- Организована двухэтапная процедура обмена валют:
  1) ENDPOINT рассчитывает сумму обмена, которая содержит сумму в целевой валюте, курс обмена и id возможной сделки. В конце первого этапа создается сделка в статусе PENDING;
  2) По id нашей сделки мы можем либо подтвердить операцию, то есть присвоить ей статус CONFIRMED,  либо отклонить со статусом REJECTED.
Одной из ключевых функциональностей является возможность формирования отчетов по операциям. ENDPOINT возвращает следующие данные за указанный пользователем период:
- Валюта;
- Сумма поступления;
- Сумма списания;
- Количество подтвержденных операций;
- Необязательный фильтр по валюте.
Еще одной функциональной возможностью данного решения является возможность выводить все НЕзафиксированные сделки, тоесть со статусом PENDING.
